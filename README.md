# ぺろんちょOS

個人専用のAI秘書アプリです。画面・基本動作に加え、OpenAI Responses APIによるタスク判断と実行前の確認画面を実装しています。

## クラウド保存・プッシュ通知（v0.6）

- メールリンクでログイン（パスワード不要）
- Supabaseへのクラウド保存と端末間同期
- 初回ログイン時に、この端末の既存タスクを引き継ぎ
- iPhone PWAへのWeb Push通知
- 期限3日前・前日・当日の毎朝通知
- 朝8時の通知、通知テスト、重複送信防止

### 初回セットアップ

1. Supabaseで新しいプロジェクトを作る
2. SQL Editorで `supabase/schema.sql` を実行する
3. `npx web-push generate-vapid-keys` でVAPID鍵を作る
4. Vercelへ `.env.example` に記載した環境変数を登録する
5. 再デプロイ後、ホーム画面版の設定からログインし、通知を有効にする

`SUPABASE_SERVICE_ROLE_KEY`、`VAPID_PRIVATE_KEY`、`CRON_SECRET` は公開せず、
Vercelの環境変数だけに保存してください。

## ホーム画面アイコン（v0.5）

- iPhone用 `apple-touch-icon.png`
- PWA用192×192・512×512アイコン
- Web App Manifest
- iPhoneでホーム画面追加した際のスタンドアロン表示

## 定例タスク（v0.4）

- 毎日・毎週・毎月・毎年の繰り返し設定
- 作成日を過ぎてからアプリを開いた場合の自動作成
- 定例タスクIDと対象期間による重複防止
- サブタスク、優先度、通知日、土日祝の期限調整を複製
- 定例の停止・削除後も、作成済みの通常タスクは保持
- 設定画面の「翌月分をテスト」から、翌月分の生成を確認可能

初期設定として「実績を計算する」と「有給休暇を確認する」が登録されます。既存の通常タスクや完了履歴はそのまま引き継がれます。

## 安全上の注意

OpenAI APIキーはコードやGitHubへ保存せず、Vercelの環境変数 `OPENAI_API_KEY` に登録します。

AIは新規タスク、サブタスク、期限変更、優先度変更、完了を提案します。曖昧な場合は確認質問を返し、ユーザーが確認画面で実行するまでデータを変更しません。

## 開発用コマンド

```bash
npm install
npm run dev
```

定例タスクの自動テストは次のコマンドで実行できます。

```bash
npm run test:recurring
```
